선박 안전장비 관리 시스템 (Ship Safety Equipment Management System)

프로젝트 소개
이 미니 프로젝트는 선박 내의 각종 안전장비를 체계적으로 관리하기 위한 시스템을 설계하고 구현하는 것을 목표로 합니다.  
선박에는 구명뗏목, 소화기, 비상 탈출 장비, 구명조끼 등 다양한 안전장비가 탑재되어 있으며,  
이 장비들은 정기 점검 주기,  유효기간, 보관 위치, 상태 등을 지속적으로 관리해야 합니다.
본 프로젝트는 이러한 데이터를 디지털화하여 선박 운영자가 효율적으로 안전장비 상태를 확인하고,  
정기 점검 및 교체 일정을 관리할 수 있도록 지원하는 프로그램입니다.

 주요 기능
- 안전장비 등록 / 수정 / 삭제 / 조회 기능  
- 정기 점검일 및 유효기간 관리
- 보관 위치 및 장비 상태 표시 (정상 / 점검필요 / 교체필요)
- 장비별 점검 이력 기록 기능
- 점검 주기 도래 시 알림 기능(추가 개발 가능) 

개발 환경
- 사용 언어 : Python (또는 PPT 기준 언어에 맞게 조정 가능)  
- 개발 도구 : Visual Studio Code / Eclipse  
- 데이터베이스 : SQLite 또는 CSV 파일 기반 저장  
- 시각화 : matplotlib, pandas (장비 점검 통계나 상태 분석 시 활용 가능)

프로젝트 목표
- 선박 안전장비 데이터의 체계적 관리 시스템 구현 
- 데이터 구조 및 객체지향 설계 원칙을 실제 시스템에 적용  
- 현장 실무에 응용 가능한 시스템 설계 능력 향상

프로젝트 진행 계획 
1단계 : 주제 선정 및 요구사항 정의 
2단계 : 데이터 설계(ERD) 및 기능 정의 
3단계 : 기본 CRUD 기능 구현 
4단계 : 점검 주기 및 알림 기능 추가 
5단계 : 테스트 및 보고서 작성
