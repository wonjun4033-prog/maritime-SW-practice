20220570 허원준 해양SW설계실습 8번째 과제

선박 안전장비 관리 시스템 (Ship Safety Equipment Management System) 최종 보고서

1. 프로젝트 개요
선박 내 안전장비(구명뗏목, 소화기, 구명조끼, 비상탈출장비 등)는 선원과 승객의 생명과 직결되는 필수 요소로, 체계적인 관리가 반드시 필요합니다. 본 미니 프로젝트는 이러한 안전장비의 정보를 디지털화하고, 점검 주기 및 상태를 효율적으로 관리할 수 있는 시스템을 설계하는 것을 목표로 합니다.

2. 요구사항 정의

 주요 기능

 안전장비 등록, 수정, 삭제, 조회 기능 제공
 장비별 정기 점검일 및 유효기간 관리
 보관 위치 및 현재 상태(정상/점검필요/교체필요) 표시
 장비별 점검 이력 기록 기능 제공
 점검 예정일 도래 시 알림 기능 포함(추가 개발 가능)

사용자 요구사항

 선원 또는 관리자 계정으로 로그인 가능
 모든 안전장비를 리스트 형태로 관리하기 쉽게 조회 가능
 점검이 필요한 장비를 자동으로 표시
 점검 이력은 날짜/점검자/점검결과를 포함

3. ER(Entity-Relation) Diagram 설명

주요 엔티티

1. Equipment(안전장비)

   EquipmentID (PK)
   Name
   Type
   Location
   ExpirationDate
   Status

2. Inspection(점검 이력)

   InspectionID (PK)
   EquipmentID (FK)
   InspectionDate
   Inspector
   Result

3. User(관리자/선원)

   UserID (PK)
   Name
   Role

주요 관계
Equipment 1 : N Inspection
User 1 : N Inspection

4. Use Case Diagram 요약

 주요 액터

   관리자(Admin)
   선원(User)

 주요 유스케이스

   장비 등록하기
   장비 수정하기
   장비 삭제하기
   장비 조회하기
   점검 이력 기록하기
   점검 예정 알림 확인하기


 5. UI(User Interface) 설계

주요 화면 구성

1. 로그인 화면

    ID/PW 입력 후 관리자 또는 사용자 권한 확인

2. 메인 대시보드

    전체 장비 수, 점검 필요 장비, 유효기간 임박 장비 표시

3. 안전장비 리스트 화면

    정렬/필터 기능
    장비 상태 컬러 태그 표시(녹색/노란색/빨간색)

4. 장비 상세 정보 화면

    장비 기본 정보
    점검 이력 리스트
    상태 변경 및 수정 버튼

5. 점검 등록 화면

    점검자, 날짜, 점검결과 입력


6. 시스템 설계 및 구조

본 프로젝트는 객체지향 설계 원칙(OOP)에 따라 각 엔티티를 클래스로 정의하고, CRUD 기능 구현을 중심으로 구조화 했습니다.

클래스 구성

Equipment 클래스: 등록/수정/삭제/상태 확인
Inspection 클래스: 이력 저장, 점검 결과 관리
DatabaseManager 클래스: SQLite 또는 CSV 저장 관리
UIManager 클래스: 사용자 인터페이스 처리

7. 개발 환경

언어: Python
IDE: Visual Studio Code
DB: SQLite 또는 CSV
라이브러리: pandas, matplotlib(통계/시각화)


8. 기대 효과

* 선박 안전장비 관리의 체계화 및 디지털 전환 가속
* 점검 누락 방지로 인한 안전성 강화
* 장비 이력 관리 자동화로 업무 효율 상승
* 실제 선박 현장에서 활용 가능성 높음


9. 결론

본 프로젝트를 통해 안전장비 관리의 중요성을 이해하고, 요구사항 분석부터 ERD, 유스케이스, UI 설계까지 전 과정을 경험했습니다. 또한 실제 시스템 개발에 필요한 데이터 구조 및 설계 능력을 향상시킬 수 있었으며, 선박 실무에서도 적용 가능한 시나리오 기반 시스템을 완성할 수 있었습니다.
